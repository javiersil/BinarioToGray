# BinarioToGray
Convertidor de  Binario a Gray  
   tienes que tener en cuenta que para convertir de binarios a Gray los valores que deben ser sumados en base 2 
   sin tomar encuenta el acarreo.    
   1+1=0
   0+0=0
   1+0=1
   0+1=1
